# mistral-qlora-craic
Fine-tuning Ministral-3-3B-Base-2512 using QLoRA for Irish-English dialect adaptation.

## Requirements
- Nvidia CUDA GPU
- Python 3.13

## Setup Guide
Create a compatible virtual environment:
- python -m venv .venv
- .venv/Scripts/activate
- pip install -r requirements.txt
