# mistral-qlora-craic
Fine-tuning Mistral 7B v0.3 using QLoRA for Irish-English dialect adaptation.

## Requirements
- Nvidia CUDA GPU
- Python 3.13

## Setup Guide
Create a compatible virtual environment:
- python -m venv .venv
- .venv/Scripts/activate
- pip install -r requirements.txt
